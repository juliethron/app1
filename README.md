# Simple Noroff Social Media App

A minimal frontend that connects to the [Noroff API](https://docs.noroff.dev/docs/v2).

## Features
- Login with your Noroff social account
- View all posts
- Create and delete your own posts
- Uses ES6 modules
- Deployed via GitHub Pages / Netlify / Vercel

## How to Run
1. Clone this repository
2. Open `index.html` in your browser
3. Or deploy via Netlify / GitHub Pages
